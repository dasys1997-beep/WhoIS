# BookMind

Telegram Mini App для читачів детективів та інших книг з великою кількістю персонажів.

## Версія 1.2 — що реалізовано

- Список книг з групуванням по жанрах, видалення книги
- Персонажі книги з ролями, видалення персонажа
- Швидкий запис персонажа, додавання тексту до існуючого
- Картка персонажа з табами: Огляд / Події / Нотатки
- Прогрес читання, 4 теми оформлення
- Дані зберігаються в Supabase
- **Голосовий запис замість фото** — говориш про персонажа, AI розбирає сказане на сегменти (характеристика/подія), пропонує кожен окремо для підтвердження
- **Лічильник згадувань** — скільки разів відкривав картку персонажа
- **Статус активності** — зелений/сірий індикатор (активний/вибув), перемикається тапом

## Налаштування Supabase

1. supabase.com → New Project
2. **SQL Editor** → виконати `supabase-schema.sql` (для нової бази) АБО якщо база вже існує — по черзі виконати:
   - `supabase-migration-001-events.sql`
   - `supabase-migration-002-mentions-status.sql`
3. **Settings → API Keys** → скопіювати **Project URL** і **Publishable key**
4. На Vercel: **Environment Variables** → `VITE_SUPABASE_URL`, `VITE_SUPABASE_ANON_KEY`

## Налаштування Gemini (AI: голос + майбутні фічі)

1. ai.google.dev → Get API key
2. На Vercel: **Environment Variables** → `GEMINI_API_KEY` (без префіксу VITE_)

## Важливо про мікрофон

Голосовий запис вимагає дозволу на мікрофон у браузері. Telegram Mini Apps зазвичай запитують це автоматично при першому натисканні "Почати запис" — якщо запис не стартує, перевір дозволи мікрофона в налаштуваннях Telegram (Settings → Privacy → ...) або системних дозволах браузера.

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на Vercel

Push у GitHub → автоматичний редеплой.

## Підключення до Telegram бота

@BotFather → `/mybots` → бот → **Bot Settings** → **Menu Button** → URL з Vercel.

## Технічний стек

- React 19 + Vite
- Supabase (база даних)
- Gemini API через Vercel Serverless Function (`/api/recognize-voice.js`)
- MediaRecorder API (запис голосу в браузері)
- Іконки: Tabler Icons, шрифти: Spectral + Inter
