# BookMind

Telegram Mini App для читачів детективів та інших книг з великою кількістю персонажів.

## Версія 1.3 — що реалізовано

- Список книг з жанрами, картки персонажів з ролями
- Швидкий запис персонажа, додавання тексту до існуючого
- Картка персонажа з табами: Огляд / Події / Нотатки
- **Редагування характеристики, подій та імені персонажа** напряму (іконка олівця)
- Голосовий запис з розбором на сегменти (характеристика/подія) через Gemini AI
- Лічильник згадувань, статус активності (зелений/сірий)
- **Свайп ліворуч → червона кнопка видалити** на картках книг і персонажів (замість кнопки-кошика)
- **Кнопка "Дочитано"** на картці книги → форма рецензії (рейтинг 1-10 + текст) → книга переходить в архів
- **Окремий екран Архів** — дочитані книги, можна повернути назад в активне читання або видалити назавжди
- **Тема зберігається між сесіями** (Supabase, а не тільки в пам'яті браузера)
- 4 теми оформлення

## Налаштування Supabase

Якщо база вже існує — виконати по черзі нові міграції:
- `supabase-migration-001-events.sql`
- `supabase-migration-002-mentions-status.sql`
- `supabase-migration-003-archive-theme.sql`

Для нової бази — просто `supabase-schema.sql` повністю.

## Жест свайпу

Тягнеш картку книги/персонажа ліворуч — з'являється червона зона з кошиком. Тап на червону зону видаляє одразу, без додаткового підтвердження (бо сам жест свайпу вже є усвідомленою дією). Невеликий натяк "‹‹" з правого краю картки підказує про можливість свайпу тим, хто ще не пробував.

## Запуск локально

```bash
npm install
npm run dev
```

## Деплой на Vercel

Push у GitHub → автоматичний редеплой. Заголовки `Cache-Control: no-cache` на index.html запобігають застарілому кешуванню в Telegram WebView.

## Підключення до Telegram бота

@BotFather → `/mybots` → бот → **Bot Settings** → **Menu Button** → URL з Vercel.

## Технічний стек

- React 19 + Vite
- Supabase (база даних, включно з налаштуваннями теми користувача)
- Gemini API через Vercel Serverless Function (`/api/recognize-voice.js`)
- MediaRecorder API (запис голосу)
- Власний хук `useSwipeToDelete` (touch + mouse events, без зовнішніх бібліотек)
- Іконки: Tabler Icons, шрифти: Spectral + Inter
