# BookMind

Telegram Mini App для читачів детективів та інших книг з великою кількістю персонажів. Допомагає не загубитись у тому хто є хто.

## Версія 1 — що реалізовано

- Список книг з групуванням по жанрах
- Персонажі книги з ролями (Головний, Підозрюваний, Жертва, Свідок, Другорядний — або власна роль)
- Швидкий запис нового персонажа під час читання
- Функція "додати до існуючого" — коли опис персонажа розкидано по кількох місцях книги
- Нотатки книги та персонажа
- Прогрес читання (номер розділу)
- 4 теми оформлення: Пергамент, Ніч, Ліс, Еспресо

## Запуск локально

```bash
npm install
npm run dev
```

Відкриється на http://localhost:5173

## Деплой на Vercel

1. Залити цей репозиторій на GitHub
2. На vercel.com → "Add New Project" → обрати репозиторій
3. Vercel автоматично визначить налаштування з vercel.json
4. Натиснути Deploy
5. Отримати URL типу `https://bookmind-xxx.vercel.app`

## Підключення до Telegram бота

1. Відкрити [@BotFather](https://t.me/BotFather)
2. `/mybots` → обрати бота → **Bot Settings** → **Menu Button**
3. Вставити URL з Vercel
4. Готово — кнопка меню в боті відкриває застосунок

## Технічний стек

- React 19 + Vite
- Іконки: Tabler Icons (CDN)
- Шрифти: Spectral (заголовки) + Inter (текст), Google Fonts
